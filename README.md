## ConsisRec
Code for paper "ConsisRec: Enhancing GNN for Social Recommendation via Consistent Neighbor Aggregation"


## How to use:

bash train.sh

## Requirements

* python 3.8
* torch 1.7.1
* numpy  1.20.0
* sklearn
